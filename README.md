     1. Production Studio name
     2.Film genre 
     3.Name of production manager
     4.Name of the movie
     5. Production costs
     6.Duration of the movie. Know that:
         -If <60p: Display "Less than 1 hour"
         -If >60p and <120p: Display "Less than 1 hour"
